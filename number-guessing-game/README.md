# 🎲 Number Guessing Game

A web version of the classic console number guessing game! Guess a random number between 1-100 in 6 attempts.

## 🎮 How to Play
1. Click **New Game**.
2. Enter your guess (1-100) and submit.
3. Get hints: Too high 📈 or Too low 📉.
4. Win in attempts ≤6 or lose after 6 tries.
